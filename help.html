Help Center

Find answers to frequently asked questions about ProjectVault.

How do I add a project?

Click the New Project button on your dashboard and complete the project form.

Can I edit a project?

Yes. Open one of your projects and select the Edit option to update your information.

How do I delete a project?

Open the project and select Delete. You will be asked to confirm before removing it.

What information should I include?

Include:

  • Project Name
  • Description
  • Technologies Used
  • GitHub Repository
  • Optional Screenshot

Need more help?

Contact the ProjectVault team for additional assistance.
