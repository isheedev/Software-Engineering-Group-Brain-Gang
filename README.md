# ProjectVault

## About the Project
ProjectVault is a web application that helps students organize and showcase their software projects in one place. Users can store project details such as descriptions, technologies used, and GitHub links.

---

## Features
- Create and manage user accounts  
- Add and edit projects  
- Store project descriptions and technologies  
- Search and filter projects  
- Save project data in a database  

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap  
- Backend: Node.js, Express.js  
- Database: MySQL
